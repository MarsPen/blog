### 问题 
不同二叉搜索树 <a>https://leetcode.cn/problems/unique-binary-search-trees/</a>

### 回答

- 思路: dp
- 时间复杂度：O(n2)
- 空间复杂度：O(n)

```js
// 代码
let numTrees = function (n) {
  let dp = new Array(n+1).fill(0)
  dp[0] = 1
  for(let i = 1; i <= n; i++) {
    for (let j = 1; j <= i; j++) {
      dp[i] += dp[j-1] * dp[i-j]
    }
  }
  return dp[n]
}
```
